# reports/issues

This directory contains publicly published **issue completion reports** for the AT Medical ecosystem.

## Naming Convention

All reports in this directory follow the ATMED naming convention:

```
ATMED-ISSUE-[AI]-[YYYY]-[XXXX].md
```

| Segment | Description                                      | Example      |
|---------|--------------------------------------------------|--------------|
| `ATMED` | Organisation prefix (fixed)                      | `ATMED`      |
| `ISSUE` | Report type (fixed for this directory)           | `ISSUE`      |
| `[AI]`  | AI system that generated the report              | `COPILOT`    |
| `[YYYY]`| Four-digit year of report generation             | `2026`       |
| `[XXXX]`| Four-digit zero-padded sequential ID             | `0001`       |

## Examples

- `ATMED-ISSUE-COPILOT-2026-0001.md`
- `ATMED-ISSUE-COPILOT-2026-0002.md`

## Contents

Issue completion reports document the successful resolution of GitHub Issues within AT Medical repositories, including:

- Executive summary
- Changes made
- Validation results
- Security considerations
- Next recommendations
