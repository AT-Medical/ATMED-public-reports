# reports/prs

This directory contains publicly published **pull request completion reports** for the AT Medical ecosystem.

## Naming Convention

All reports in this directory follow the ATMED naming convention:

```
ATMED-PR-[AI]-[YYYY]-[XXXX].md
```

| Segment | Description                                      | Example      |
|---------|--------------------------------------------------|--------------|
| `ATMED` | Organisation prefix (fixed)                      | `ATMED`      |
| `PR`    | Report type (fixed for this directory)           | `PR`         |
| `[AI]`  | AI system that generated the report              | `COPILOT`    |
| `[YYYY]`| Four-digit year of report generation             | `2026`       |
| `[XXXX]`| Four-digit zero-padded sequential ID             | `0001`       |

## Examples

- `ATMED-PR-COPILOT-2026-0001.md`
- `ATMED-PR-COPILOT-2026-0002.md`

## Contents

PR completion reports document the successful merging of pull requests within AT Medical repositories, including:

- Summary of changes introduced
- Review outcomes
- Test and validation results
- Post-merge status
