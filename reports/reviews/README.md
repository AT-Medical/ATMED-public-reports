# reports/reviews

This directory contains publicly published **review reports** for the AT Medical ecosystem.

## Naming Convention

All reports in this directory follow the ATMED naming convention:

```
ATMED-REVIEW-[AI]-[YYYY]-[XXXX].md
```

| Segment  | Description                                      | Example      |
|----------|--------------------------------------------------|--------------|
| `ATMED`  | Organisation prefix (fixed)                      | `ATMED`      |
| `REVIEW` | Report type (fixed for this directory)           | `REVIEW`     |
| `[AI]`   | AI system that generated the report              | `COPILOT`    |
| `[YYYY]` | Four-digit year of report generation             | `2026`       |
| `[XXXX]` | Four-digit zero-padded sequential ID             | `0001`       |

## Examples

- `ATMED-REVIEW-COPILOT-2026-0001.md`
- `ATMED-REVIEW-COPILOT-2026-0002.md`

## Contents

Review reports document structured assessments and audits of AT Medical repositories, codebases, or processes, including:

- Review scope and methodology
- Findings and observations
- Compliance status
- Recommendations
